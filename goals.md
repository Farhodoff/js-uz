# JS 1 oylik reja

Har hafta 4-6 soat ajrating. Har band qisqa amaliyot bilan yakunlansin.

## Hafta 1: Asoslar
- [ ] Kirish va muhit (browser, console, script ulash) - amaliyot: 3 marta console.log.
- [ ] O'zgaruvchilar (var/let/const, nomlash) - amaliyot: 5 ta qiymat saqlash.
- [ ] Ma'lumot turlari va typeof - amaliyot: 6 ta tipni tekshirish.
- [ ] Operatorlar va shartlar (if/else, ternary) - amaliyot: 5 shartli misol.
- [ ] Looplar (for, while) - amaliyot: 1..100 yig'indisi.
- [ ] Funksiya asoslari (param, return) - amaliyot: 3 funksiya yozish.

## Hafta 2: Massiv va ob'ektlar
- [ ] Massivlar (push/pop/length/index) - amaliyot: 5 shahar ro'yxati.
- [ ] Array metodlari (map/filter/forEach/reduce) - amaliyot: 3 ta transform.
- [ ] Ob'ektlar (property, nested) - amaliyot: user profil ob'ekti.
- [ ] Destructuring (object/array) - amaliyot: qiymatlarni ajratish.
- [ ] Spread/Rest (...) - amaliyot: massiv va ob'ekt birlashtirish.
- [ ] Arrow function va callbacklar - amaliyot: map bilan ishlatish.

## Hafta 3: DOM va asinxronlik
- [ ] DOM tanlash va o'zgartirish (querySelector, textContent, classList) - amaliyot: 3 element.
- [ ] Event listenerlar (click, input, submit) - amaliyot: counter.
- [ ] Forma va oddiy validatsiya - amaliyot: bo'sh maydon tekshirish.
- [ ] Promise asoslari (then/catch/finally) - amaliyot: 2s timeout.
- [ ] Async/await - amaliyot: timeout kutish.
- [ ] Fetch API (GET, JSON) - amaliyot: public API dan ma'lumot olish.

## Hafta 4: Modul, xatolar, loyiha
- [ ] Modullar (import/export) va fayl tuzilma - amaliyot: 2 modul.
- [ ] Xatolar bilan ishlash (try/catch, throw) - amaliyot: validate funksiya.
- [ ] Debugging (devtools, breakpoint, console) - amaliyot: bug topish.
- [ ] LocalStorage (set/get, JSON) - amaliyot: sozlamani saqlash.
